# Edgaze (Production Baseline)

Production-grade baseline for the Edgaze web app (Next.js 15, App Router, TypeScript, Tailwind).

## Commands

```bash
# Install deps
pnpm install   # or npm install / yarn install

# Run dev server
pnpm dev       # http://localhost:3000

# Type-check
pnpm typecheck

# Lint
pnpm lint

# Build
pnpm build

# Start (prod)
pnpm start
