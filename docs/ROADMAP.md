# Roadmap

This document outlines the planned features and future direction for Edgaze. Items are subject to change based on priorities and feedback.

---

## Current version: 1.0

**Status:** Production ready

The platform is live with core functionality:
- ✅ Prompt Studio with placeholders and versioning
- ✅ Workflow Builder with visual canvas
- ✅ Marketplace with curated discovery
- ✅ Authentication (email/password + Google OAuth)
- ✅ Run tracking and analytics
- ✅ Profile pages and handle system

## Long-term vision

### Community & ecosystem

- **Plugin marketplace** — Third-party node types and integrations
- **Developer SDK** — Build custom integrations and tools
- **Community hub** — Forums, tutorials, and resources
- **Certification program** — Official Edgaze creator certification

### Intelligence & automation

- **AI-assisted building** — Suggest nodes and connections
- **Auto-optimization** — Optimize workflow performance automatically
- **Anomaly detection** — Alert on unusual run patterns
- **Smart recommendations** — Personalized content discovery

### Scale & performance

- **Edge deployment** — Deploy workflows to edge locations
- **Real-time collaboration** — Multiple users editing simultaneously
- **Advanced caching** — Faster run execution
- **Distributed execution** — Run workflows across multiple regions

## How we prioritize

Features are prioritized based on:

1. **User impact** — How many users benefit?
2. **Strategic value** — Does it align with our vision?
3. **Feasibility** — Can we build it well with current resources?
4. **Dependencies** — Does it enable other features?
5. **Community demand** — What are users requesting?
